== iFlexiload App

Its a RoR based application for loading recharge amount from online to Cellphone.

= Features

1. User can load amount without ot with login to their account.
2. Loggedin User can preload balance to their account.
3. Loggedin User can view load history
4. Loggedin User get Notification on short of account balance
5. Loggedin User can list all his contact Numbers to their account.
